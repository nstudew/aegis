# Aegis 🛡️

Кроссплатформенный менеджер паролей для Windows, macOS и Linux.
Всё шифрование выполняется локально, данные никогда не покидают устройство.

## Возможности

- Локальное шифрование: мастер-пароль -> Argon2id -> ключ -> AES-256-GCM. Мастер-ключ не хранится на диске и затирается в памяти при блокировке.
- Генератор паролей: длина 8-64, регистры/цифры/символы, исключение похожих символов, оценка стойкости в битах энтропии.
- Крипто-кошельки: отдельный тип записи с сид-фразой (12/24 слова), адресом и сетью. Сид-фраза показывается нумерованной сеткой.
- TOTP (2FA): генерация одноразовых кодов прямо в записи, с кольцом обратного отсчёта.
- Проверка на утечки (HIBP) по модели k-anonymity: на сервер уходит только первые 5 символов SHA-1, сами пароли не передаются.
- Аудит: поиск слабых и повторяющихся паролей.
- Иконки сайтов (favicon) у записей.
- Автоблокировка по таймауту бездействия и при сворачивании окна.
- Авто-очистка буфера обмена: скопированный пароль стирается через заданное время.
- Импорт/экспорт зашифрованного бэкапа и импорт CSV из Bitwarden и Chrome/Edge.
- Горячие клавиши: Ctrl+F поиск, Ctrl+N новая запись, Ctrl+L блокировка, Ctrl+Shift+C копировать пароль.

## Установка

### Windows
1. Скачайте `Aegis_x.x.x_x64-setup.exe` из раздела Releases.
2. Запустите установщик. Приложение не подписано платным сертификатом, поэтому SmartScreen может показать предупреждение - нажмите "Подробнее" -> "Выполнить в любом случае".
3. Готово, Aegis появится в меню "Пуск" и на рабочем столе.

Альтернатива - `Aegis_x.x.x_x64_en-US.msi` для корпоративного развёртывания.

### macOS
1. Скачайте `Aegis_x.x.x_aarch64.dmg` (Apple Silicon) или `Aegis_x.x.x_x64.dmg` (Intel).
2. Откройте образ и перетащите Aegis в "Программы".
3. Первый запуск: правый клик по иконке -> "Открыть" (приложение не нотаризовано).

### Linux
- AppImage: `chmod +x Aegis_x.x.x_amd64.AppImage && ./Aegis_x.x.x_amd64.AppImage`
- Debian/Ubuntu: `sudo dpkg -i Aegis_x.x.x_amd64.deb`

## Первый запуск

1. Придумайте мастер-пароль - он шифрует всё хранилище. Его невозможно восстановить: если забыть, данные потеряны.
2. Добавляйте записи (логины или крипто-кошельки) кнопкой "+".
3. Регулярно делайте экспорт зашифрованного бэкапа (Настройки -> Хранилище).

Файл хранилища лежит в каталоге данных приложения ОС:
- Windows: `%APPDATA%\com.aegis.passwords\vault.aegis`
- macOS: `~/Library/Application Support/com.aegis.passwords/`
- Linux: `~/.local/share/com.aegis.passwords/`

## Безопасность

| Слой | Реализация |
|------|------------|
| Деривация ключа | Argon2id, 64 MiB, 3 итерации |
| Шифрование | AES-256-GCM, свежий nonce на каждое сохранение |
| Хранение ключа | только в RAM, zeroize при блокировке |
| Запись на диск | атомарная (temp + rename) |
| Сеть | только HIBP (k-anonymity) и favicon; пароли наружу не уходят |

## Сборка из исходников

Требуется Node 18+ и Rust 1.77+.

```bash
npm install
npm run app:dev      # запуск в режиме разработки
npm run app:build    # сборка установщика под текущую ОС
```

Linux дополнительно требует системные пакеты WebKitGTK (см. документацию Tauri).

Сборка установщиков под все три ОС автоматизирована в GitHub Actions
(`.github/workflows/release.yml`): поставьте тег `vX.Y.Z` и запушьте,
готовые установщики появятся в Releases.
